    Install a Apache Server on Vultr

    VPS的探索

    通过对VPS的建设，我了解了大概的后端建设。虚拟服务器相当于一台假设在海外的电脑，我们需要使用ssh来控制服务器。

    shadowSocket

    服务器可能会有几种操作系统，我了解的是cent OS6。通过在终端的ssh操作，我们可以用命令行来操控服务器系统。

    ssh root@heyao.art

    先用ssh登录服务器的根目录，上一段代码输入后还需要输入服务器的密码，这个密码是由Vultr服务商提供的。复制一下就好了。

    yum -y install wget

wget -N --no-check-certificate https://raw.githubusercontent.com/ToyoDAdoubi/doubi/master/ssr.sh && chmod +x ssr.sh && bash ssr.sh

    运行这个脚本来安装shadowsocket，设置好以后，服务器就能提供ss服务了。再到电脑或手机上安装可用的ss客户端。

    Web服务器

    用几种服务器脚本能使服务器提供网页给浏览器。 Apache是一种经典的服务器程序，在服务器上安装并启动Apache后，便可以在浏览器上通过输入ip地址来访问服务器了。但是还是有几件事需要去做。

    首先要通过修改配置文件来修改服务器的默认访问路径。因为我们要使用ftp服务来方便的上传和部署网页文件在服务器上。首先先要安装ftp的程序在服务器，要记住账号和密码，以后登录的时候需要用到。

    用WebStorm或者别的ftp软件把网页上传在服务器上，就能通过ip访问网页啦。

    如果有域名的话，在域名的设置界面把服务器的ip填上去，就能通过域名来浏览服务器了。
