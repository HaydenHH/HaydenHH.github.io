    Pattern export

    After finished the logical stage, I can get accurate data of Chinese Character, so I can generate graphic by these data. The after question is that, how can I form these words.

    Methodology of generation

    1. Translate each words to shape.
    2. Translate whole sentence, random choose some elements to build graphic.
    3. collect data and choose some of them to build whole typing words.

    Attribute of graphic

    To build a group of graphic, each attribute of that should be defined by data which collected from user typing. I did a mindmap to clarify that:

    Graphic

    Abstract

    abstract shape like geometry graphic can represent some feeling or emotion that hard to describe, and it can be generate quickly, efficiently and easily.

    Object

    specific shape can be used to represent the radicals of character, because it can show real objects they are. So this attr might be supplied by Radical.

    Fill

    This attribute decide how this shape be filled and drawn.

    color

    textrue

    Texture might be generated by bitmap or Noise

    pattern

    Here I drew some pattern by AI, but in later project, it could be generated directly by Javascript, which support interaction with user.

    Form

    position

    symmetry

    transform

    compose

    This is a error, but it has strong visuals because of repetition and growth

    Content of project

    The original idea is showing visualisation of CC, by actually I can’t convey absolutely right information of word. Therefore, I might need an other form to express this project. A story? or A location which user can generate. For summary, user should get more aspiration to type rather than type something random.
