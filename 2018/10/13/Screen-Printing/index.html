    Screen Printing

    The induction of screen printing is really interesting for me. Though I contacted screen printing before, heres tutor gave us many useful information.

    Steps to print

    • The first step is clear the table and prepare a laysheet. That’s the paper we would print as outcome. And we must bring original work. We should place the original work to comfirm the position we print on laysheet. So we paste work (it generally is acid paper) Fixed by tape. Than we fix film board on print machine so it hard to move. A little tip is fold a paper and paste on laysheet, so it is available to move and adjust to a right position though the sheet will be covered by film board.

    • The ink should be adjust to be wet and smooth that’s means it will not dry too quick. To confect (调制) a good quality ink, we should balance the ratio of acrylic medium and acrylic. Medium should be put into container at first, because it can available acrylic mixed completely. 50% medium and about 10g acrylic, and tiny other colours if we nee d mixed colour. 其实就挤三四陀就好了。

    • To print well, we should print ink equality, DO NOT print overly, it lose all of details of engraving. The most important thing is that if we print a multicolour work, the position of laysheet should be accuracy place. We use first work that took mark with laystops to guarantee that.

    Why use screen printing?

    Screen printing is a technique that print something have original details. Though it could print repeatedly, every times printing with different speeds or force might make different effect.

    Furthermore, the making of film board could be hand drawn that’s means some hand drawn illustrations can be use as film, and print to different colors. Different with digital creation, the process of making film could have tiny differences between original painting, so it will full of unpredictable details that really gorgeous for me.
