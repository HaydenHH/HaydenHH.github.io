    The arts of science

    We are both rely on technology and smart devices today, and digital art or digital media have been really popluar in public sight. As designers, we generally use computer everyday, it is hard to imagine that how can I work when I leave computer. Did computer really support us to make incredible work? Is it REALLY VITAL THING in creation process? Or we merely rely on convient drawing or storage? It could make sence to understand early computer work and relieze how early computer artists thinking about art.

    In the early computer generated work, most of work could be made by mathematician or scientists, because they should operate complex progremmer on computer. Sometimes they did works with random shape or regular form, which could not be defined to an art work. If we focus on their process of working, we can find that they are trying to make computer to create something that generally repeat and change with some variable.

    As Manfred Mohr said,

    it is more important that develop a priori vocabulary which integrates the computer into the aesthethic system.

    As a tool, computer can make really fine image with actually a easy way. The more difficult thing is that how can we make computer create image as a CREATOR? That’s means computer should have the ability to create image that does’t create by hand. In fact, computer would ever create something from ZERO. It must opearte with some programmer that set by artist. In Manfred’s work, I saw some shape grouped with some specific method. These shape will be no sense if anyone out from group, and if they set in a matrix, we can see the visual grammer of work. Artist set a way that computer can create polyline and circle, and the quantity of point or length of line could be random, so every shape will have tiny differents but shows amazing variety.

    In this work, letters are printed as a strict format, like frame in frame. Computer always render some info from data bank by specific method, and the exsit of data could be visualization. Analyze the attribute or quantity of data, and render to more clear visual information. For illustration, artist could use tiny elements to make a bigger group, we collect different essential elements to imitate some situation and environment.
